# Improved Messenger

## Description
End-to-end encrypted, anonymous, p2p messenger, open sourced - EncryPost by ./cosmic

## Installation
To compile the project, use the following command:

```bash
g++ -o /messenger.exe /client.cpp /mnemonic_utils.cpp -lws2_32
```
